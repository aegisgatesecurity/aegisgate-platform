MVP v1.0.0
Dashboard Certificates Settings
Note: Settings are configured via environment variables. Changes require a proxy restart to take effect.

Proxy Configuration

The address and port where AegisGate listens for connections. Set via AEGISGATE_BIND_ADDRESS environment variable.
The backend server URL that AegisGate proxies to. Set via AEGISGATE_UPSTREAM environment variable.

Security Settings

Maximum number of requests allowed per minute from a single client. Set via AEGISGATE_RATE_LIMIT environment variable. Default: 100
Maximum allowed size for request bodies (e.g., 10MB, 1GB). Set via AEGISGATE_MAX_BODY_SIZE environment variable. Default: 10MB
Maximum time allowed for processing a request. Set via AEGISGATE_TIMEOUT environment variable. Default: 30s

TLS & Certificates

Directory where TLS certificates are stored. Set via AEGISGATE_CERT_DIR environment variable. Default: ./certs

Logging

Logging verbosity level (debug, info, warn, error). Set via AEGISGATE_LOG_LEVEL environment variable. Default: info
Restart Required: To apply any configuration changes, you must restart the AegisGate proxy. Use the Dashboard to stop and restart the proxy.
