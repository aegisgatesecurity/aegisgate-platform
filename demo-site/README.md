# Demo Site - AegisGate Security Platform

HTML files for demo.aegisgatesecurity.io

## Files

- `index.html` - Main landing page
- `demo.html` - Interactive terminal demo
- `styles.css` - Shared stylesheet
- `demo.js` - Terminal emulator logic

## Deployment

1. Upload all files to your web host
2. Ensure DNS for demo.aegisgatesecurity.io points to your server
3. SSL is handled by Let's Encrypt on your host

## Features

- Responsive design (mobile-friendly)
- Interactive terminal emulator
- Mermaid diagrams
- Dark/light theme
- Docker command copy-to-clipboard
